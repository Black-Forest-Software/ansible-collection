# Ansible Collection

Collection of modules, playbooks and roles for the INWX api. This plugin is on Ansible Galaxy: https://galaxy.ansible.com/inwx/collection

## Installation

Via ansible galaxy:
```
ansible-galaxy collection install inwx.collection
```
